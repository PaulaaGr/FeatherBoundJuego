FEATHERBOUND
Videojuego de aventuras y plataformas en 3D
Descripción

Featherbound es un videojuego de aventura y plataformas en 3D donde el jugador controla a Pickle, un cuervo que debe rescatar a sus amigos atrapados en jaulas encantadas por una fuerza oscura que corrompe un bosque mágico. El juego combina exploración, mecánicas ágiles de movimiento, combate y una narrativa emotiva para crear una experiencia desafiante y atmosférica.

Características principales
Ambientación

El juego presenta un estilo visual estilizado, con escenarios ubicados en un bosque mágico y diseñados para fomentar la exploración.

Personaje principal: Pickle

Las mecánicas disponibles incluyen saltar, doble salto, volar o planear, correr, caminar, agacharse y realizar diferentes tipos de ataques.

Enemigos

El juego cuenta con cuatro enemigos principales: Draeven, Gallus, Ghost y Oruguitas, cada uno con comportamientos y ataques diferenciados.

NPCs

Incluye cuatro personajes no jugables con los que el jugador puede interactuar.

Audio

El apartado sonoro está compuesto por música atmosférica y efectos de sonido ambientales y de interacción.

Tecnologías utilizadas

El proyecto se desarrolla en Unreal Engine 5.4 y utiliza Blueprints y Animation Blueprints.

Requisitos previos

Para abrir y ejecutar el proyecto se recomienda usar Unreal Engine 5.4, Windows 10 u 11, una GPU compatible con DirectX 12, entre 8 y 16 GB de RAM y, opcionalmente, Git para clonar el repositorio.

Instalación y ejecución
Instalación

Para instalar el proyecto, primero se debe clonar o descargar el repositorio:

git clone https://github.com/PaulaaGr/FeatherBoundJuego.git


Luego, se debe abrir el archivo Featherbound.uproject en Unreal Engine 5.4 y esperar a que se compile el contenido inicial.

Ejecución

Una vez cargado el proyecto, se puede iniciar la partida presionando Play dentro del editor.

Estructura del proyecto

La estructura principal del contenido incluye carpetas de Animation, Audios, bluePrints (que contiene subcarpetas como Actors, con AtaquesPickle, Enemigos —Draeven, Gallus, Ghost y Oruguitas—, Generales —Nivel1, Nivel2, Nivel3, Nivel4 y NivelTesting— y overlap), así como Config, FunctionLibraries, InteractiveActor, Pickle_tamaño, saveSystem y Widgets. Además, existe un nivel adicional ubicado en el mapa predeterminado de Unreal Engine dentro de ThirdPersonMap.
